しろくろ

Ruby 1.9で作りました。
Twitterのクライアントです。
oauth, rubytter, tkが必要。
Windowsでしか動作しないと思います。

【使いかた】
1.アプリ認証
　"アプリ認証"をおしたら認証ページにいくのでアプリ連携を許可してください。
2.PINの入力
　アプリ連携をした後に表示される7桁の数字をテキストボックスに入力して、"PIN認証"ボタンをおす。
3.つぶやく
　テキストボックスに文字を入力して"ついーと"ボタンをおす。


"ホモ"ボタンはおしたら"┌（┌ ＾o＾）┐ﾎﾓｫ…"とつぶやきます。

【認証後】
認証をしたら同じディレクトリに"tokens"というフォルダができます。
そのフォルダの中の"tokens.txt"というファイルにはアクセストークンの情報があひってます。
もしアカウントをかえたいのならそのファイルを消した後に、別のアカウントでアプリ連携を許可してください。
そのファイルでアカウント情報を保持するのでご注意を。
